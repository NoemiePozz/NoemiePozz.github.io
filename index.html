Master projects

Ray Tracing for Global Illumination
in Video Games

This is a study about Ray Tracing and other rendering techniques (rasterisation, path tracing, radiosity), focused on the new technologies that were recently created to use real-time ray tracing in video games.

3D Multi-Target Tracking
Using Multiple Views

(Matlab)

The project focuses on projecting the position of different human targets in a 3D world. The purpose is to avoid occlusion of the persons by using several cameras of the same scene, so we can keep track of every target.

3D Reconstruction Using
Kinect Cameras

(Python & OpenCV)

Using two different views and a depth map, a 3D reconstruction of the scene was performed.

Stereoscopic Images Process

(C++ & OpenCV)

An anaglyph image was created from stereoscopic images.

Visualisation of a 3D map

(Three.js)

A map was designed with 3D objects showing activities/movements in the city.

Image Enhancement
Perceptual Evaluation

(In progress)

The purpose of this study is to analyse the impact of vividness or saturation modifications on the preference and the naturalness of an image. The results are obtained via a perceptual experiment.
